# Web AR Master Skill

This skill encompasses the complete knowledge required to build, deploy, and maintain advanced AR web applications.

## 🏗 Knowledge Domains
1. **Multi-Mode AR**: Mastering WebXR hit-testing, iOS native Quick Look integration, and gyroscope fallbacks.
2. **Pedagogical i18n**: Implementing dynamic language switching with strict RTL (Right-to-Left) support for Middle Eastern locales.
3. **Premium Design Patterns**: Applying Glassmorphism and Pure Black aesthetics for high-end user experiences.
4. **DevOps for AR**: Configuring server headers and MIME types to ensure compatibility with mobile OS restrictions.

## 📜 Source Documentation
The full architectural blueprint and commented source code are available at:
[https://github.com/bmwecker/xrbackup.git](https://github.com/bmwecker/xrbackup.git)

## 🛠 Usage Instructions
When initiating a new AR project:
1. Initialize the monolithic structure found in `core/index.html`.
2. Apply the headers from `config/_headers`.
3. Load 3D assets (GLB and USDZ) and rely on the built-in normalization logic.
